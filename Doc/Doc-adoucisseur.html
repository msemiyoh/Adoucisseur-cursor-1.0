1. Fonctionnement général


Schéma simplifié d’un adoucisseur Fleck SXT avec affichage, boutons et vanne[1]

1.1 Présentation des boutons et icônes


Bouton de régénération

Boutons de réglage
  • Bouton de régénération : Déclenche une régénération manuelle.
  • Boutons de réglage : Pour ajuster l’heure, les valeurs et les paramètres.
  • Indicateur de service : vanne en service (icône allumée)
  • Régénération programmée la nuit : icône clignotante
  • Autres : indicateur diagnostic/erreur, indicateur programmation, débit, multiplication par 1 000

1.2 Modes de régénération

  • Régénération chronométrique (tc) : Selon un intervalle de jours préréglé.
  • Régénération hebdomadaire (dAY) : Jours pré-définis dans la semaine.
  • Régénération volumétrique retardée (Fd) : Selon volume d’eau utilisé, démarrage à l’heure programmée.
  • Régénération volumétrique immédiate (FI) : Dès que la capacité est épuisée.

1.3 Affichage en service

Mode Affichage alternatif
Volumétrique Heure du jour <-> Volume restant
Chronométrique Heure du jour <-> Nombre de jours restant
Vanne Twin Heure du jour <-> Volume restant <-> bouteille en service

Exemple d’affichage digital simplifié (mode service & volumétrique)[1]

Pour régler l’heure :
1. Maintenir le bouton + ou - jusqu’à l’icône TD.
2. Régler l’heure avec +/-, valider avec le bouton de régénération.

1.4 Affichage pendant une régénération

  • BW (Backwash) : détassage
  • BD (Brine draw) : aspiration & rinçage lent
  • RR (Rapid rinse) : rinçage rapide
  • BF (Brine fill) : renvoi d'eau

Le temps restant s’affiche en minutes ; le nom du cycle clignote à son démarrage.

1.5 Démarrage d’une régénération manuelle

  • Appuyer puis relâcher le bouton de régénération : l’icône clignote, départ à l’heure programmée.
  • Appuyer et maintenir 5 s pour un départ immédiat.

Exemple d’action sur bouton de régénération[1]

1.6 Coupure de courant

  • Les données sont stockées (mémoire non volatile), l’écran éteint, les événements différés.
  • Lors du retour du courant, l’horloge clignote.
  • Le volume d’eau utilisé pendant la coupure n’est pas enregistré.

2. Programmation

2.1 Accès au mode programmation

  1. Régler l’heure à 12:01 puis sortir.
  2. Appuyer 5 secondes sur + et - simultanément (icône programmation).
  3. Appuyer sur le bouton de régénération pour naviguer, +/- pour modifier.
  4. Passer sur toutes les étapes et revenir en service pour valider.

2.3 Gestion des erreurs (codes ER)

Code Type Cause Solution
0 Capteur came Changement de cycle trop lent Vérifier moteurs, connexions, engrenages
1 Position cycle Cycle imprévu Vérifier configuration, déclencher régénération manuelle
2 Régénération Pas de régénération depuis 99j Lancer régénération manuelle, vérifier compteur
3 Mémoire Défaillance mémoire électronique Réinitialiser/programmer; contacter support si persiste

Réinitialisation :
Partielle : en mode service, maintenir + et -.
Totale : débrancher, maintenir bouton régénération au re-branchement.

3. Mode diagnostic

  1. Appuyer 5 secondes sur + et - depuis le service.
  2. Boutons +/- pour faire défiler les affichages :
  • FR : Débit instantané
  • PF : Débit de pointe
  • HR : Heures depuis régénération
  • VU : Volume utilisé depuis régénération
  • RC : Capacité de réserve
  • SV : Version carte
